Instructions

Vibify is a Spotify playback controller for Windows. Follow these steps to get started:

  1. Click the "Download Vibify" button above to download Vibify.exe.
  2. Save the file to a folder (e.g., Desktop).
  3. Double-click Vibify.exe to launch the app. If you see a "Windows protected your PC" warning, click "More info" and "Run anyway".
  4. A browser window will openâ€”log in with your Spotify account.
  5. Close the browser tab after seeing "Login Complete!"
  6. Start playing a track in Spotify, and Vibify will display the track info.
  7. Use the buttons to control playback (play, pause, skip, etc.).

Requirements: Windows, Spotify account, internet connection, Spotify app running.
